# Laptop Price Predictor

An end-to-end Machine Learning web application that predicts laptop prices based on hardware configurations like CPU, GPU, RAM, OS, and storage.

🔗 **Live Demo:** https://laptop-priceepredictor.onrender.com/

## 🚀 Features & Engineering
* **Feature Engineering:** Implemented custom categorical feature processing and feature extraction.
* **Model Pipeline:** Evaluated and optimized multiple algorithms (Linear Regression, Random Forest, and XGBoost) to achieve the best prediction accuracy.
* **Full-Stack Integration:** Built a responsive HTML/CSS frontend connected to a Python Flask backend framework to handle real-time user inputs and model inference.

## 🛠️ Tech Stack
* **Language:** Python
* **Machine Learning:** Scikit-Learn, Pandas, NumPy
* **Backend Framework:** Flask
* **Frontend:** HTML, CSS
